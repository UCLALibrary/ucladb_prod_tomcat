Wildcard

Use the question mark ? to truncate; at least 3 characters required.

africa? finds africa, african, Africana

It can be used with Keyword, Title, Journal Title, and Subject searches:

Journal math? fluid? mech?

 

For Title (start of), Journal Title (start of), Author and Title List, Subject List (start of), It can only be used to truncate the search:

Doors of per?

Phrase

Use quotation marks " " to search for an exact string of words:

"world peace"

Can not be used in Title Phrase or Call Number searches.

Quick Limits

Use to limit to English; journals; books; music scores; sound recordings; visual materials; recent years.

Can not be used with List searches.

Set Other Search Limits

Use to limit by date; location; language; format; place of publication.

Set Other Search Limits before entering search term(s).

Limits are retained between searches until cleared by the user.

Can not be used with List searches.

Keyword

Use to search for names, titles, subjects, etc.

Multiple terms are searched using an implied "AND" then results are sorted by Author (primary sort), Title (secondary sort), Date Published (tertiary sort):

        john irving cider house finds john AND irving AND cider AND house

Title

Use for title searches on works of any kind

Multiple terms are searched using an implied "AND" then search results are sorted by Author (primary sort), Title (secondary sort), Date Published (tertiary sort):

       nest flew cuckoo's finds nest AND flew AND cuckoos

Title (start of)

Use for title searches on works of any kind.

Enter exact beginning of a title; it is acceptable to either include or exclude initial articles (A, An, The, L', etc.) Search results are sorted by Title (primary sort), Author (secondary sort):

        journal of personality and social psychology
        painted bird

the painted bird
        moonlight sonata

Title Phrase

Use to search for an exact string of words in a title of any kind. Search results are sorted by Author (primary sort), Title (secondary sort), Date Published (tertiary sort):

         also rises
         wireless networks
         cell biology

Author List

Use as a keyword search for authors, editors, composers, directors, organizations, etc. Enter individual names as lastname firstname:
       austen jane

Or, if you only know part of a name enter what you know in any order:

eri? sat?
Enter organizations or groups as they appear or as keywords:

        american psychological association

association psychological American

Subject List

Use as a keyword search of terms from controlled subject vocabulary lists,
(e.g. Library of Congress Subject Headings or Medical Subject Headings)
Enter the subject or topic, omitting all punctuation. Search results are sorted by Author (primary sort), Title (secondary sort), Date Published (tertiary sort):

copy? history?

afghan? wom?

object oriented

Command Search

Use quotes to search phrases: "artificial intelligence"

Combine search terms with Boolean operators: and, or, not.

Use parentheses to group terms in complex searches: (residence or home or housing) and (elderly or aged)

Use ? with partial words to find variant endings: africa? finds africa, african, etc.

Use field codes to refine your search: hkey cl and tkey algebra

retrieves materials held in the College Library and containing the word Algebra in the title.

 

EndNote

Use to export records into EndNote.

Import Option:

Save records using the EndNote format option. Import into EndNote using the UCLA Import Filter.

Connect Option:

Connect to the Catalog using the UCLA Connection File.

EndNote version 9 includes the UCLA Import Filter and Connection File. Users of earlier versions can download the UCLA files directly from EndNote.
