WebVoyáge Help Print Authority Record 

To Print or Save the displayed catalog record:

  1. Next to "Print Format" make sure that a format is selected.
  2. Select the Press to Click to Print button
  3. A new screen will display containing only a printable/saveable version of the record.
  4. Print using the browser's Print function, or
  5. Save using the browser's Save As function (supply a new filename when prompted).
