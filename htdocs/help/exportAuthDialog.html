It is possible to Export records in the MARC Format.
These formatted records may then be imported into any MARC based catalog system, but require special software to view.
Three versions of MARC are available: Latin-1, non-Unicode/MARC-8 and Unicode/UTF-8.

  • Latin 1 -
    provides a MARC-formatted record utilizing the ASCII character set
  • Raw Marc (non-Unicode) -
    provides a MARC-formatted record utilizing the ASCII character set plus special characters not available in ASCII.
  • UTF-8 MARC (Unicode) -
    provides a MARC-formatted record according to the Universal Coded Character Set (UCS or ISO/IEC 10646) and its industry subset Unicode that have been approved for the standard interchange of MARC records. This standard permits the display of non-Roman characters.
